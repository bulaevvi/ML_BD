# "Машинное обучение на больших данных". ДЗ №5 - Линейная регрессия Breeze + Spark ML  

При написании проекта использовалась ОС Win10. Настройку окружения и установку IDEA выполнял в соответствии с тем, 
как это было продемонстрировано Андреем Кузнецовым и Дмитрием Киселевым на лекциях 5 и 6 текущего курса. С выполнением 
предыдущих заданий особых проблем не было (см. репозиторий и https://data.vk.company/blog/topic/view/19515/), однако в ДЗ №5 возникли сложности:  
1. С `sparkVersion = "3.0.1"` проект не собирался. Путем проб и ошибок удалось пофиксить проблему, указав `sparkVersion = "3.2.0"`  
2. Так и не смог запустить тесты, связанные с чтением-записью модели на диск. Перепробовал много рецептов из интернета, но конкретно в моем случае 
они не помогли. Выдавалась вот такая ошибка:  
`
An exception or error caused a run to abort: org.apache.hadoop.io.nativeio.NativeIO$POSIX.stat(Ljava/lang/String;)Lorg/apache/hadoop/io/nativeio/NativeIO$POSIX$Stat; 
java.lang.UnsatisfiedLinkError: org.apache.hadoop.io.nativeio.NativeIO$POSIX.stat(Ljava/lang/String;)Lorg/apache/hadoop/io/nativeio/NativeIO$POSIX$Stat;
`

Примечание:  
- Выявление "скрытой модели" сделал в одном из тестов, отмеченным комментарием:  
`Нагрузочный тест для проверки правильности обучения "скрытой модели"`  
- Тесты, связанные с чтением/записью модели, реализовал, но в связи с вышеописанными проблемами запустить их локально не смог.
